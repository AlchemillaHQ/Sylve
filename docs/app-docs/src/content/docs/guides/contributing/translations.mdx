title: Translations
description: Contribute localization updates for Sylve.
---

Sylve uses regular `.po` files for translations.

We use a library called [Wuchale](https://wuchale.dev/), but translators usually do not need to work with it directly.

## Where Translation Files Live

Translation files are in:

```bash
web/src/locales/
```

## Translation Workflow

- Open the relevant `.po` file in `web/src/locales/`.
- Update translations and review them carefully for accuracy and consistency.
- Run any project checks your workflow requires.
- Open a pull request with your updated `.po` files.

If possible, include a short note in your pull request about the language and sections you updated.

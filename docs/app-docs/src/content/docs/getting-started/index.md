Welcome to Sylve. This guide helps you get from zero to a working installation quickly.

## What Sylve Is

Sylve is a lightweight FreeBSD control panel for managing VMs, Jails, and ZFS workflows from a single interface.

## Prerequisites

Before you begin, make sure you have:

- A FreeBSD environment where you can install and run Sylve.
- Network access to reach the machine from your browser.
- Basic shell access for installation and setup commands.

## Install

Install dependencies for this docs site locally:

```bash
npm install
```

Then start the docs site:

```bash
npm run dev
```

## Your First Run

Open the docs in your browser:

- `http://localhost:4321/docs`

From there, continue with the first practical walkthrough:

- [Example Guide](/getting-started/example/)

## Verify It Works

You should see:

- The Getting Started page loading at `/docs`.
- Sidebar navigation for the `getting-started` section.
- The Example Guide opening successfully.
